# Dearlog

Hazel 전용 생성형 학습 + 메타인지 + 칭찬 강화 MVP입니다.

## 구성 파일
- `index.html` : 메인 화면
- `styles.css` : 스타일
- `app.js` : 로직(localStorage 기반)

## 주요 기능
- 과목 직접 추가/삭제
- 과목별 미션 직접 추가/삭제
- 학습 세션 기록
  - 입력
  - 구조화
  - 생성
  - 검증
  - 메타인지 전/후 기록
- 칭찬 엔진
- 아카이브 검색/필터
- 간단한 분석 대시보드

## 사용 방법
1. `index.html`을 브라우저에서 열기
2. 과목과 미션을 원하는 대로 추가
3. 학습 세션에서 기록 남기기
4. 대시보드/분석에서 패턴 확인

## 저장 방식
- 현재는 `localStorage`에 저장됩니다.
- 브라우저 데이터를 삭제하면 기록도 삭제될 수 있습니다.
