# Dearlog v2

로컬 퍼스트 기반 생성형 학습 웹앱입니다.

## 포함 기능
- 과목 직접 추가/삭제
- 과목별 미션 직접 추가/삭제
- 학습 세션 기록
  - 입력
  - 핵심 구조화
  - 생성 결과
  - 검증 결과
  - 메타인지 전/후 기록
- 칭찬 엔진
  - 기본 모드
  - 가나디 모드
- 최근 세션 / 아카이브 검색
- 데이터 내보내기 / 불러오기 / 초기화
- 간단한 분석 요약

## 실행 방법
1. 압축 해제
2. `index.html`을 브라우저에서 열기

## 저장 방식
- `localStorage`
- 백업은 JSON export/import 방식
